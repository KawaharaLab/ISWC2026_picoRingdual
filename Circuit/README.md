# Hardware Circuit Design

This directory contains the hardware design and PCB layout files for the *picoRing dual* system. 

## Directory Structure

* **`ring/`**: Schematic and PCB design files for the  ring.
* **`wristband/`**: Schematic and PCB design files for the wristband.

---

## Design Environment

* **Software**: **KiCad (v7.0 or later)**
* **Project Files**: `.kicad_pro`

To inspect or modify the designs, open the respective `.kicad_pro` file inside the `ring/` or `wristband/` directory using KiCad.

---

## Important Notice for Replication

* **Component Footprints & Libraries**: Custom or third-party footprint/symbol libraries used in these designs are not bundled in this repository. Users must set up, map, or substitute their own component libraries and footprints according to their local KiCad environment and parts availability.
